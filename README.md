# IS-IS_Protocol_for_IPv6
Đồ án môn học: Các giao thức truyền thông
- Vận hành mô hình mạng Dual-stack (IPv4 \& IPv6) với cấu trúc phân cấp định tuyến (Level 1, Level 2, Level 1-2) trên phần mềm GNS3 và hệ điều hành Cisco IOS.
- Cấu hình định tuyến Multi-Topology và tích hợp cơ chế bảo mật xác thực HMAC-MD5 nhằm ngăn chặn rủi ro giả mạo, đảm bảo tính toàn vẹn của dữ liệu định tuyến.
- Sử dụng công cụ Wireshark để bắt gói tin (Packet Capture), qua đó phân tích chuyên sâu cấu trúc bản tin IS-IS PDU và các trường TLV mở rộng.
- Soạn thảo báo cáo kỹ thuật chi tiết và xây dựng slide thuyết trình, chứng minh hệ thống đạt trạng thái hội tụ hoàn toàn và truyền tải thông suốt gói tin đầu cuối.
- Công cụ sử dụng: GNS3, Cisco IOS, Wireshark
